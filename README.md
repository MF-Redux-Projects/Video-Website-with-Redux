#### Project Features:

- [x] Video list
- [x] Video details
- [x] Related videos
- [x] Video pagination
- [x] User can search by title
- [x] User can filter video by clicking on category
- [x] User can filter video by clicking on author
- [x] User can clear filter

#### Packages Use
- [x] [React](https://reactjs.org/)
- [x] [Redux](https://redux.js.org/)
